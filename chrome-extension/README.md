# 🦣🍩 Token Optimizer Chrome 확장

ChatGPT, Claude, Gemini에서 프롬프트를 자동 압축. **토큰 70%+ 절약**.

## 기능

- 🦣 ChatGPT/Claude/Gemini 입력창 자동 감지
- ⚡ 한 클릭으로 즉시 압축
- ⌨️ 단축키: `Ctrl+Shift+C`
- 🌍 한국어 + 영어 모두 지원
- 🎚️ 3단계 강도: Lite / Full / Ultra
- 📊 실시간 토큰 통계

## 설치 방법 (개발자 모드)

1. Chrome에서 `chrome://extensions/` 열기
2. 우측 상단 **"개발자 모드"** ON
3. **"압축해제된 확장 프로그램을 로드합니다"** 클릭
4. 이 폴더 (`chrome-extension`) 선택
5. 끝! 🎉

## 사용법

### 방법 1: 페이지 내 버튼
- ChatGPT/Claude 접속
- 프롬프트 작성
- 우측 하단 **🦣 압축** 버튼 클릭

### 방법 2: 단축키
- 입력창에 텍스트 작성
- `Ctrl+Shift+C` 누르기

### 방법 3: 확장 팝업
- 브라우저 우측 상단 확장 아이콘 클릭
- 텍스트 붙여넣기 → 압축 → 복사

### 방법 4: 우클릭 메뉴
- 어떤 페이지든 텍스트 선택
- 우클릭 → **🦣 Caveman으로 압축**

## 지원 사이트

- chatgpt.com

## 압축 강도

| 강도 | 특징 | 압축률 |
|------|------|--------|
| **Lite** | 군더더기만 제거, 문법 유지 | 30-50% |
| **Full** | 관사·주어 생략 (권장) | 50-75% |
| **Ultra** | 전보체, 최대 압축 | 70-90% |
