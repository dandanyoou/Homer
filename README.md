# 🦣 Caveman - 프롬프트 압축기

AI 도구의 토큰 낭비를 줄이세요. 긴 프롬프트를 짧고 간결한 "Caveman 말투"로 자동 압축합니다.

## 문제

- AI 도구(ChatGPT, Claude 등) 사용자들이 불필요한 설명으로 토큰을 낭비
- 같은 의미를 훨씬 짧게 표현 가능
- 결과적으로 비용 증가, 응답 시간 지연

## 솔루션

**Caveman**은 규칙 기반 압축으로:
- 불필요한 존댓말 제거 ("~습니다" → 생략)
- 불필요한 접속사 제거 ("그리고", "그래서" 등)
- 마크다운 형식 정리
- 중복 문장 제거
- 공백 최적화

## 사용법

1. [dancaveman.github.io](https://dandanyoou.github.io/dancaveman/) 열기
2. 긴 프롬프트 붙여넣기
3. "압축하기" 클릭
4. 압축된 프롬프트 복사

## 예시

**원본 (156 토큰):**
```
안녕하세요. 저는 지금 여러분의 도움이 필요합니다. 
제가 만들고 있는 웹사이트의 디자인을 개선해야 하는데요, 
특히 사용자 인터페이스(UI)를 더욱 직관적이고 아름답게 만들고 싶습니다. 
그리고 모바일 환경에서도 잘 보일 수 있도록 반응형 디자인을 적용하고 싶습니다.
```

**압축 (48 토큰):**
```
웹사이트 UI 개선. 직관적, 예쁘게. 반응형 디자인 필요.
```

**압축률: 69% 절약** ✅
